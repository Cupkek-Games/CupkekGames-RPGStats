# CupkekGames RPGStats

Character stats / attributes / modifiers system. Standalone — usable without inventory.

## What's inside

- **`RPGStats/`** (CupkekGames.RPGStats.asmdef) — base stat types, attribute definitions, modifier system, leveling support.

## Dependencies

- `com.cupkekgames.core`
- `com.cupkekgames.luna` (UI integration)
- `com.cupkekgames.data` (stat data via IData)

## Installation

Embedded package — clone alongside the other CupkekGames packages.

## Related packages

- `com.cupkekgames.inventory` — inventory system; integrates with RPGStats for equipment-stat modifiers via the `InventorySystem.RPGStats` bridge
